10-Examen

Examen: creacion de biomaterial y molde

Objetivo

Nueuestro objetivo consiste en diseñar un molde para construir un objeto con un biomaterial, el cual tambien sera creado por nosotros. Nuestra intencion es crear un objeto innovador y sustentable que a la vez sea util para otro proyecto que tiene directa relacion con agua y plantas. Es por esto que decidimos diseñar un mole que nos permita crear un objeto que cumpla la funcion de maceta germinadora, este objeto en forma es similar a los clasico metodos que utilizan para germinar como : cajas de huevos o cajas con compartimientos.

La ventaja de crear una maceta germinadora con la clasica forma pero con nuestro biomaterial es que facilita el proceso de plantacion ya que no seria necesario transplantar y evitaria el riesgo de muerte que las plantas sufren en aquel metodo y ademas agregaria abono a la tierra una ves se deshace el germinador con el agua de riego o lluvia.

Resultado

Conclusion

El resultado de esta segunda experimentacion con este biomaterial es satisfactoria ya que ahora el material es mas compacto, no se quiebra a pesar de los golpes, es un material similar al cemento. La diferencia con el primer experimento, es que a este se le agrego materia organica: hojas de arbol. Esto permitio que el material se uniera de mejor forma, quedando aun mas resistente y evitando que se fracture. Tambien se pudo experimentar la resistencia al agua del biomaterial, este no es resistente al agua ya que al humedecerse comienza a deshacerse, esto nos llamo la atencion y lo vimos como una oportunidad para crear el molde con este biomaterial.
